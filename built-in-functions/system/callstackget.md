# CallStackGet

Returns an array of structs by default of the current tag context.

## Method Signature

```
CallStackGet(maxFrames=[numeric])
```

### Arguments

| Argument    | Type      | Required   | Description   | Default   |
| ----------- | --------- | ---------- | ------------- | --------- |
| `maxFrames` | `numeric` | `false`    |               | -1        |
| ----------  | ------    | ---------- | ------------- | --------- |

## Examples

```
CallStackGet(maxFrames=[numeric])
```
