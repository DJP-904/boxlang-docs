# IsDateObject

Determine whether a given value or variable reference is a date or dateTime object.

## Method Signature

```
IsDateObject(value=[any])
```

### Arguments

| Argument   | Type   | Required   | Description                        | Default   |
| ---------- | ------ | ---------- | ---------------------------------- | --------- |
| `value`    | `any`  | `true`     | Value to test for date object-ness |           |
| ---------- | ------ | ---------- | -------------                      | --------- |

## Examples

```
IsDateObject(value=[any])
```
