# ArrayFind

Return int position of value in array, case sensitive

## Method Signature

```
ArrayFind(array=[array], value=[any])
```

### Arguments

| Argument   | Type    | Required   | Description               | Default   |
| ---------- | ------- | ---------- | ------------------------- | --------- |
| `array`    | `array` | `true`     | The array to be searched. |           |
| `value`    | `any`   | `true`     | The value to found.       |           |
| ---------- | ------  | ---------- | -------------             | --------- |

## Examples

```
ArrayFind(array=[array], value=[any])
```
