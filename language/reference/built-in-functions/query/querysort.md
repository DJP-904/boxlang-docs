# QuerySort

Sorts array elements.

## Method Signature

```
QuerySort(query=[query], sortFunc=[function])
```

### Arguments

| Argument   | Type       | Required   | Description          | Default   |
| ---------- | ---------- | ---------- | -------------------- | --------- |
| `query`    | `query`    | `true`     | Query to sort        |           |
| `sortFunc` | `function` | `true`     | Sort function to use |           |
| ---------- | ------     | ---------- | -------------        | --------- |

## Examples

```
QuerySort(query=[query], sortFunc=[function])
```
