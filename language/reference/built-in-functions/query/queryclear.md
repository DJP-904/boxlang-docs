# QueryClear

This function clears the query

## Method Signature

```
QueryClear(query=[query])
```

### Arguments

| Argument   | Type    | Required   | Description        | Default   |
| ---------- | ------- | ---------- | ------------------ | --------- |
| `query`    | `query` | `true`     | The query to clear |           |
| ---------- | ------  | ---------- | -------------      | --------- |

## Examples

```
QueryClear(query=[query])
```
