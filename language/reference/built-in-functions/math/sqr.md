# Sqr

Returns the square root of a number

## Method Signature

```
Sqr(value=[numeric])
```

### Arguments

| Argument   | Type      | Required   | Description                             | Default   |
| ---------- | --------- | ---------- | --------------------------------------- | --------- |
| `value`    | `numeric` | `true`     | The number to return the square root of |           |
| ---------- | ------    | ---------- | -------------                           | --------- |

## Examples

```
Sqr(value=[numeric])
```
