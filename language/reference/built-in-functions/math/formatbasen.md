# FormatBaseN

Converts a number to a string representation in the specified base.

## Method Signature

```
FormatBaseN(number=[numeric], radix=[integer])
```

### Arguments

| Argument   | Type      | Required   | Description   | Default   |
| ---------- | --------- | ---------- | ------------- | --------- |
| `number`   | `numeric` | `true`     |               |           |
| `radix`    | `integer` | `true`     |               |           |
| ---------- | ------    | ---------- | ------------- | --------- |

## Examples

```
FormatBaseN(number=[numeric], radix=[integer])
```
