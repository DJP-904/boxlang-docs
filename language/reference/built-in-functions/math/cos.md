# Cos

Returns the cosine of an angle entered in radians

## Method Signature

```
Cos(number=[numeric])
```

### Arguments

| Argument   | Type      | Required   | Description                                         | Default   |
| ---------- | --------- | ---------- | --------------------------------------------------- | --------- |
| `number`   | `numeric` | `true`     | The number to calculate the cosine of (in radians). |           |
| ---------- | ------    | ---------- | -------------                                       | --------- |

## Examples

```
Cos(number=[numeric])
```
