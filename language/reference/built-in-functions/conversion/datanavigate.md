# DataNavigate

Constructs a fluent data navigator based on different types of data.

## Method Signature

```
DataNavigate(data=[any])
```

### Arguments

| Argument   | Type   | Required   | Description   | Default   |
| ---------- | ------ | ---------- | ------------- | --------- |
| `data`     | `any`  | `true`     |               |           |
| ---------- | ------ | ---------- | ------------- | --------- |

## Examples

```
DataNavigate(data=[any])
```
