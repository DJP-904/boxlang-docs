# DollarFormat

Formats a number as a U.S.

## Method Signature

```
DollarFormat(number=[any])
```

### Arguments

| Argument   | Type   | Required   | Description                                   | Default   |
| ---------- | ------ | ---------- | --------------------------------------------- | --------- |
| `number`   | `any`  | `true`     | The number to format as a U.S. Dollar string. |           |
| ---------- | ------ | ---------- | -------------                                 | --------- |

## Examples

```
DollarFormat(number=[any])
```
