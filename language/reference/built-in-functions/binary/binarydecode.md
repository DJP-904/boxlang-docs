# BinaryDecode

Encodes binary data to a string with the specified algorithm

## Method Signature

```
BinaryDecode(string=[string], encoding=[string])
```

### Arguments

| Argument   | Type     | Required   | Description   | Default   |
| ---------- | -------- | ---------- | ------------- | --------- |
| `string`   | `string` | `true`     |               |           |
| `encoding` | `string` | `true`     |               |           |
| ---------- | ------   | ---------- | ------------- | --------- |

## Examples

```
BinaryDecode(string=[string], encoding=[string])
```
