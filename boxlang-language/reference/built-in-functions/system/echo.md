# echo

Print a message with line break to the buffer

## Method Signature

```
echo(message=[any])
```

### Arguments

| Argument  | Type  | Required | Description          | Default |
| --------- | ----- | -------- | -------------------- | ------- |
| `message` | `any` | `true`   | The message to print |         |

## Examples

```
echo(message=[any])
```
